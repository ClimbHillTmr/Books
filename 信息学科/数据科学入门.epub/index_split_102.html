对于多重回归分析来说，参数向量通常称为 β 。我们希望这个向量也包括一个常数项，为此，只要向我们的数据中添加一列即可：


beta = [alpha, beta_1, ..., beta_k]



同时：


x_i = [1, x_i1, ..., x_ik]



那么，我们的模型可以用下列函数实现：


def predict(x_i, beta):
    """assumes that the first element of each x_i is 1"""
    return dot(x_i, beta)



就本例而言，自变量 x 是一个向量型列表，每个列表元素如下所示：


[1,    # 常数项
